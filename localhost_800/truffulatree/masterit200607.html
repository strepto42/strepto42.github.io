4/7/06


My father in-law is doing the grey nomad thing in his bertha-sized 'bago, and wants Internet access. One option is to attach a satellite dish, but would his 'bago need to be even more bertha? Who provides this service, and how many arms and legs might it cost, or would you recommend some other option?

Satellite dishes are a popular way of receiving TV on the road, but the Internet side of things makes for quite a different animal.

Most commonly, satellite connections use a phone line for uploads, which means it isn't of much use when travelling. Two way satellite is available, but it's a fixed service that needs professional installation, so it's not a travel option (and even if it was, it will also set you back several limbs).

The best option is to look at a CDMA wireless solution. Telstra, being the lumbering Hydra that it is, offers two versions of this; one through Bigpond and one through Telstra Mobile. Both cost the same and use the same network, but use different (albeit equivalent) hardware.

The Bigpond version comes with email addresses and web hosting space, whereas the Mobile version does not (it just provides the connection). The mobile version, however, will function as a CDMA mobile phone, capable of making and receiving calls and text messages, but only when it's plugged into a host computer.

Links for the two products are:

http://www.telstra.com.au/mobile/business/products/mobilebroadband.htm
http://my.bigpond.com/internetplans/broadband/wireless/mobile_plans/default.jsp

CDMA coverage is quite good (most populated areas are covered), and whilst the raw speed of the connection is nothing to write home about in rural areas (it's about twice modem speed), it does work well.

If you're in the right coverage area, you can also connect in a much faster mode (called EV-DO), which will give you broadband-like download speeds. Currently these fast coverage areas are only located in capital cities and a few other regional centres.

Telstra, of course, have a complete monopoly here, and so the value for money of the service isn't amazing (even if the pricing is nowhere near as scary as it is for satellite).

For example, Telstra's policy of charging in minimum blocks of 5 minutes is quite irritating, and it also makes it quite hard to accurately keep track of how much time you've used.

Nonetheless, in a massive country like Oz, CDMA is the best solution.


25/7/06


From time to time I come across the advice to backup my hard drive. I have Windows XP and one link advises me to go to System Tools and to use the backup function, but I don't seem to have this. What method do you recommend for backing up my hard drive? (I have a Dell Dimension 8300).

XP Pro comes with Microsoft backup installed, but XP Home doesn't. It is available though - see http://tinyurl.com/3vvzg.

Regardless of availability, Microsoft must love supporting other businesses, because Windows' backup programs have always stunk, particularly because they've never supported optical removable media or data compression.

After trying many third party backup programs over the years, I finally settled on a thing called Acronis Trueimage. It's not free, but it is excellent (both in function and reliability).

You can find it at http://www.acronis.com. The home version will set you back $US50, but there is also a trial version available so you can give it a go.

Another, very similar product that many people use is Norton Ghost. It has the almost the same feature set as Acronis, but it doesn't quite have the same reliability. In fact, it's just another rebadged product which Symantec/Norton have bought; originally it was called Powerquest v2i backup.

Symantec do this all the time, and rarely actually write their own software, which is partly why they have so many problems. If only their software engineers were as good as their marketing department, their reputation amongst IT support staff might be better.

Anyway, some key features you should look for in any good backup software include:

- the ability to backup entire disks, or just specified files and folders
- being able to run in the background automatically, to a schedule if you like
- support for removable media, as well as file-based network backups
- support for data compression

Perhaps most importantly, a good set of good disaster recovery options is also essential - if you can't restore your system there's no point in backing it up.

It's hidden away in the Internet Explorer options. Go to tools->options in IE, then select the Content tab, then click the Autocomplete button, and tick the various boxes.
