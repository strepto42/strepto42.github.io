11/3/03


I was recently using my PC when my APC brand UPS decided to go crazy and cut all power. My PC now crashes on startup just after the "Loading Windows XP" screen with a message stating "Stop: c0000218 {Registry File Failure} The registry cannot load the hive (file): \SystemRoot\System32\Config\SOFTWARE or its log or alternate". What happened? Please help! My thesis is on this PC!

Ouch, sounds nasty. The worst case scenario is that you may be looking at a complete reinstall of Windows. On the plus side though it's very likely your uni files are still intact.

What most likely happened is that when your UPS cut the power, Windows was in the process of writing to the registry, and it has been corrupted.

The first thing to recommend here is to ditch your UPS for another model. If you can't trust it to maintain power it's not much use!

Secondly, have a minor slap on the wrist for (we assume) not having vital files backed up! Even something as simple as emailing your thesis to a friend every day can save you a lot of heartache.

As far as fixing the problem goes, try this. It gets a little involved but if you take it step by step it should be fairly easy.

First, you'll need to boot from your original XP installation disc. After it finishes loading, press "R" to activate the System Recovery Console.

For help getting this far you can also refer to support.microsoft.com/default.aspx?scid=kb;en-us;q307654

Once at the recovery console prompt (most likely C:) type the following commands:

cd windowssystem32config
ren software software.bad
copy windows epairsoftware software
exit

Your PC will restart, and should not crash. Don't be alarmed if everything looks wrong and you get some minor error messages, the main thing here is that we are back into Windows.

Next, run "System Restore", located in Start->Programs->Accessories->System Tools. Follow the instructions on the screen, it's fairly straightforward. Try to pick the most recent date before the problem happened.

Finally, reboot and cross your fingers!

If you still have problems, a fresh copy of Windows is needed. But first grab a friendly geek to help you get your files - they are almost certainly still there and intact.


18/3/03


My IE keeps crashing all the time. It used to be fine but now it crashes each time after 15 minutes or so. Got any advice?

First, make sure you are up to date with the latest version of IE, and all the patches required. IE6 SP1 is the latest offering from Microsoft, and can be found at www.microsoft.com/ie.

If that doesn't help, you may need to reinstall Windows (the old favourite). Another option is to give a different browser a go.

Mozilla (www.mozilla.org) is a good place to start; it is a very stable browser, available not only for Windows but Macs and Linux too. It is also the browser that Netscape 6 and 7 are built on, but it comes with somewhat less baggage than the aforementioned behemoth.

It has a bunch of features that IE lacks, including the ability to block certain images and popup windows (in other words, ads).

The browser also has a feature called "tabbed browsing" which lets you have several web pages open in a single window, and is a must for anyone who loves opening 50 sites at once.

Additionally, Mozilla comes with a very decent mail & newsgroup reader. The latest beta version even supports a degree of automatic junk mail filtering.


More on Mac-PC networking
We've also had some feedback from readers regarding the MasterIT column that dealt with networking that-which-is-beige to that-which-is-chrome. It seems we were stuck back in the last century with the technology!

Macs running OS9 do require the Dave software to network with PCs, however Macs with OS X 10.1 and above can talk to PC networks without any trouble at all.

From the Finder, select the Connect to Server option in the Go menu. From here you can browse the network and select the PC and share you wish to connect to, it's that easy.

Users running OS X 10.2 can also share their files on a PC network in a similar way. See the following two links for more information:

docs.info.apple.com/article.html?artnum=106471 (for connecting to a PC)
docs.info.apple.com/article.html?artnum=107083 (for sharing files with a PC)


25/3/03


I'm thinking of purchasing a DVD drive for my PC, and I was wondering what the deal is with watching movies. Do I need any special software? My video card also has a TV out - I'd like to find out about using it as well. I'm using XP.

It's no problem at all playing back DVD movies on computer.

You'll need some DVD playback software, but most DVD drives these days come bundled with it.

PowerDVD is a common package, and is a very decent program - it's quite configurable and will even let you utilise surround sound (provided your sound card supports it).

Using your video card's TV-out with XP is also pretty easy. Exactly how you go about it depends on which brand of video card you have; the settings are generally tucked away in the advanced section of the Display Properties.

If you have an Nvidia brand card, you might also want to check out TVTool (www.tvtool.com) - it's not free, but gives you a lot more options for TV output than the standard drivers allow, like support for widescreen output and flicker reduction.


I've had cable internet connected for some time, but now I'd like to share it with my flatmates. What are the options? One friend said we'll need a router costing several hundred dollars but I've also heard it can be done more cheaply with just a PC running Windows.

Strictly speaking you don't need a router to share a cable connection. But it will make life a lot easier, and you won't need to leave your PC on all the time.

Most internet sharing devices are very simple to get going as well, and they double as pretty good firewalls.

Either way, first you'll need to set up a network connecting all the PCs that will want access, which is a whole other topic.

Assuming you have a working network already and don't want to invest in a router, things get a little more complex.

Rather than go into detail here, check out www.ozcableguy.com (a great resource for Australian broadband), specifically the cable guides. There are a few different ones for various types of Windows.
