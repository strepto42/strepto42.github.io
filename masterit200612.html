I took your advice regarding getting a spam filter, but while looking on one of the sites, I now have a new task bar, toolbar888 - how do I remove it?

Unfortunately toolbar888 appears to be a piece of spyware.

It's sad, but these days even benevolent programs like spam filters and their kin can turn out to be Trojans that harbour nasties like this - the very things they're supposed to remove.

Grab yourself a reputable spyware remover like Adaware (www.lavasoft.de) or Spybot Search and Destroy (www.safer-networking.org); they should make short work of the grubby little monster.


I was wondering if it's possible to make Windows run certain programs automatically, at a given time and/or interval? For example, I would like to set up an automatic defrag every month or so, to run in the middle of the night.

It's quite easy to make XP run programs automatically; you just need to become acquainted with the Task Scheduler.

It's accessed via the Control Panel. The option you are after is called "Scheduled Tasks". If you're using the newer, simplified XP control panel layout, rather than the classic layout, it's filed under "Performance & Maintenance".

Once open, double click the "Add Scheduled Task" icon, and proceed through the wizard. You'll need to tell it which program to run (by browsing to its location), and, of course, when to run it.

For once Microsoft have got it right, and you can specify a variety of schedules and options; examples include: run once only, or at startup, daily, weekly, or when your computer is idle.

Setting up a task to run a full defrag is, therefore, pretty easy. Microsoft even have full instructions for doing it yourself at tinyurl.com/4k6r8.

Bear in mind that XP already does a partial defrag from time to time when your system is idle, so you probably don't need to schedule a full defrag terribly often.

Also, it's a good idea to think about scheduling a backup before each defrag. The MS defrag is fairly safe, but nonetheless, if the power goes out in the middle of such an operation the potential for total disaster, albeit small, is still there.
